# Archive Docs

이 폴더는 현재 canonical workflow나 공식 실험 규범이 아닌 과거 초안과 보존용 계획 문서를 둔다.

여기에 있는 문서는 참고용 기록으로는 가치가 있지만, 현재 논문 본문이나 실행 규칙의 source of truth로 읽으면 안 된다.

현재 archive에는 아래 문서가 들어 있다.

- `proposals/research-plan-extreme-flood-underestimation.md`:
  설명형 상세 연구계획서 초안
- `proposals/research-proposal-submission-draft.md`:
  제출용 문체에 맞춘 연구계획서 초안
