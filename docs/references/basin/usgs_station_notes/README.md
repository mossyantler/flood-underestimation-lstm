# USGS Station Notes

이 폴더는 event suppression / managed-flow 진단을 시작하기 전에 확인한 USGS station page와 annual water-data report PDF 내용을 station별로 저장하는 곳이다. `docs/experiment/analysis/model/09_event_suppression_diagnosis_protocol.md`에서 요구하는 external station context의 local reference artifact다.

각 note는 원문을 길게 복사하지 않고, 출처 URL, 확인일, station-level remark, event 해석에 쓰인 판단만 짧게 남긴다. Annual water-data report PDF를 찾지 못했으면 `not found`로 적고, 찾은 PDF가 event year가 아니라 historical station sheet라면 그 한계도 함께 쓴다.

이 폴더의 note는 공식 data source를 대체하지 않는다. Case-level 해석에서 USGS source를 읽었는지 추적하고, 이후 같은 유역을 다시 볼 때 regulation, diversion, gage relocation, datum, rating caveat를 빠르게 확인하기 위한 보조 기록이다.
